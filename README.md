Um pequeno projeto em PHP que simula um sistema bancário executado no terminal.

O objetivo do projeto é praticar conceitos fundamentais de programação em PHP, como estruturas de repetição, condicionais, manipulação de entrada de dados via STDIN e organização de lógica de negócio.

📋 Funcionalidades

- Exibe o titular e o saldo inicial da conta
- Menu interativo com as opções:
  - Consultar saldo atual
  - Sacar valor
  - Depositar valor
  - Encerrar aplicação

✔ Validações implementadas

- Impede saque maior que o saldo disponível
- Impede saque ou depósito de valores negativos ou iguais a zero
- Exibe mensagem para opções inválidas no menu
