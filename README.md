Sistema bancário em PHP executado via terminal para prática de lógica de programação e validação de operações financeiras.

O sistema simula um pequeno ambiente bancário executado no terminal, permitindo consultar saldo, realizar saques e depósitos por meio de um menu interativo.

📋 Funcionalidades

Exibe o titular e o saldo inicial da conta
Menu interativo com as opções:
Consultar saldo atual
Sacar valor
Depositar valor
Encerrar aplicação

✔ Validações implementadas

Impede saque maior que o saldo disponível
Impede saque ou depósito de valores negativos ou iguais a zero
Exibe mensagem para opções inválidas no menu
